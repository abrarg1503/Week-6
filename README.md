## Overview
A responsive weather dashboard that fetches real-time weather and 5-day forecast using OpenWeatherMap API.

## Features
- Current weather display
- 5-day forecast
- City search
- Local Storage persistence
- Loading and error handling
- Responsive design

## Setup Instructions
1. Clone the repository
2. Replace API key in js/api.js
3. Open index.html in browser
